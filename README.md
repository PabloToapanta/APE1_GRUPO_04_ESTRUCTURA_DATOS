## Ejecucion del codigo desde la terminal
Para ejecutar el codigo desde la terminal se debe primero ejecutar el comando
``` bash
g++ nombre_del_archivo.cpp -o nombre_del_archivo
```
Luego de que se haya creado el archivo `nombre_del_archivo.exe` correrlo con el comando
``` bash
.\nombre_del_archivo.exe
```
